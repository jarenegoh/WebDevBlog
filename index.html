In Semester 2, I am starting my journey of learning HTML and CSS!!

So far, this is my progress and what I have learned in the following weeks.

WEEK 1:

  • I learned what web development is and how different HTML is from Python.
  • I learned the language of HTML and CSS and its power on websites.
  • I learned the basic elements like the header element, paragraph element, list element etc..
  • I learned how to add images and videos to my website.

WEEK 2:

  • I learned about the Semantic Web.
  • I learned common elements used in HTML5 Semantics and their functions.
  • I learned how to add Cascading Style Sheets (CSS) to style my HTML with colors and different fonts.

WEEK 3:

  • I learned how to use CSS selectors.
  • I learned about the CSS positioning property and how to move things.
  • I learned flexboxes, and how to apply flexbox to get flexible layouts.

WEEK 4:

  • I learned how to add responsive images to a website.
  • I learned about Git, which is a version control system that keeps track of files.

WEEK 5:

  • I learned how to create my own individual codespace on Github.
  • I learned how to open the Javascript console from Chrome.
  • I think my favourite part about html is the editors :)
  • I learned about strings and variables in html and how different and similar they are to strings and variables in Python.
  • I learned about how to manipulate objects and arrays.
